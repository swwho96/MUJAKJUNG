# 옷 분류 모델 만들기

## 프로젝트 소개
- Kaggle 'Clothing dataset' 데이터를 활용한 옷 분류 모델 만들기
- pytorch를 활용한 옷 분류 모델 만들기


## Tool
- `Python`
- `Pytorch`
- `Colab`

## 프로젝트 현황
- T-shirt, shorts, shoes 등 16가지 class 분류
- Custom CNN model, EfficientNet fine tuning model
- F1-score 'macro' 0.61
